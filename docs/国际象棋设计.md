# 国际象棋实现设计

## 概述

按照围棋/五子棋/中国象棋的架构模式，在 `games/chess/` 下实现完整的国际象棋游戏，包括规则引擎、AI、棋盘组件和房间页面。

## 棋盘与规则

- **棋盘**: 8列 × 8行（row 0-7, col 0-7），交替深浅色格
- **白方先行**
- **棋子**: 白方（♔♕♖♗♘♙）/ 黑方（♚♛♜♝♞♟）

### 棋子走法规则

| 棋子 | 走法 |
|------|------|
| 王 (King) | 任意方向一步（横、竖、斜） |
| 后 (Queen) | 横、竖、斜任意距离（车+象的组合） |
| 车 (Rook) | 横或竖直线任意距离 |
| 象 (Bishop) | 斜走任意距离，不能越子 |
| 马 (Knight) | 走日字（L形），可越子 |
| 兵 (Pawn) | 向前一步（初始可两步）；斜前方吃子 |

### 特殊规则

- **王车易位 (Castling)**: 王和车均未移动、中间无子、王不经过被将军的位置
  - 王翼易位: 王向右移动两格，车移到王旁边
  - 后翼易位: 王向左移动两格，车移到王旁边
- **吃过路兵 (En Passant)**: 兵从初始位置走两步后，相邻对方兵可将其吃掉
- **兵升变 (Promotion)**: 兵到达对方底线，可升变为后、车、象、马
- **将军 (Check)**: 被将军必须应将
- **将死 (Checkmate)**: 无法应将则输
- **逼和 (Stalemate)**: 无子可走但未被将军，判和
- **50步和棋**: 连续50步无吃子无兵移动，判和

## 集成点

1. `shared/src/types/game.ts` — `Chess = 'chess'`
2. `server/src/websocket/Dispatcher.ts` — `getEngine()` 添加国际象棋分支
3. `client/src/pages/RoomPage.tsx` — 添加 `chess` 路由
4. `client/src/pages/LobbyPage.tsx` — 创建房间配置 `rows: 8, cols: 8`
5. `server/src/websocket/Dispatcher.ts` — AI 走棋调度（返回 `{ from, to }`）

## 棋盘 UI 设计

- Canvas 绘制，8×8 交替深浅色格（浅色 #f0d9b5，深色 #b58863）
- 棋子用 Unicode 符号（♔♕♖♗♘♙ / ♚♛♜♝♞♟）
- 选中高亮、合法走法提示点、吃子空心圆环
- 上一步走法轨迹高亮
- 坐标标签（a-h, 1-8）
- 落子音效、将军音效、将死音效
- 白方视角在下方，黑方视角自动翻转

## AI 设计

- Minimax 搜索 + Alpha-Beta 剪枝
- 搜索深度: 1-4 层（对应四个难度）
- 评估函数: 棋子价值 + 位置价值表 + 将军威胁
- 棋子价值: 后=900, 车=500, 象=330, 马=320, 兵=100, 王=10000
- 位置价值表: 标准国际象棋评估表（兵、马、象、车、后、王各一套）

## 棋谱保存

- **先手**: 白方
- **存储位置**: `game_records` 表的 `pgn` 字段
- **棋谱格式**: PGN（Portable Game Notation）
- **格式说明**: 使用代数记谱法，棋子缩写 K/Q/R/B/N，坐标 a-h + 1-8
- **示例**: `1. e2e4 e7e5 2. Nf3 Nc6 3. Bb5`
- **特殊记法**: 王车易位 O-O（王翼）/ O-O-O（后翼）
