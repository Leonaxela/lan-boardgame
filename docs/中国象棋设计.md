# 中国象棋实现设计

## 概述

按照围棋/五子棋的架构模式，在 `games/chinese-chess/` 下实现完整的中国象棋游戏，包括规则引擎、AI、棋盘组件和房间页面。

## 棋盘与规则

- **棋盘**: 9列 × 10行（row 0-9, col 0-8）
- **红方先行**
- **棋子**: 红方（帅仕相馬車砲兵）/ 黑方（将士象马车炮卒）
- **猜先**: 石头剪刀布，赢家执红

### 棋子走法规则

| 棋子 | 走法 |
|------|------|
| 帅/将 | 横或竖一步，不能出九宫（红: row 7-9, col 3-5; 黑: row 0-2, col 3-5） |
| 仕/士 | 斜走一步，不能出九宫 |
| 相/象 | 斜走两步（田字），不能过河，不能被塞象眼 |
| 馬/马 | 走日字，不能被蹩马腿 |
| 車/车 | 横竖直线任意距离 |
| 砲/炮 | 移动同車，吃子需隔一个棋子（炮架） |
| 兵/卒 | 未过河只能向前一步；过河后可左右前，不能后退 |

### 特殊规则

- **将帅对面**: 将和帅不能在同一直线上中间无子
- **将军**: 被将军必须应将
- **将死**: 无法应将则输
- **困毙**: 无子可走则判负

## 集成点

1. `shared/src/types/game.ts` — `ChineseChess = 'chinese-chess'`
2. `server/src/websocket/Dispatcher.ts` — `getEngine()` 添加中国象棋分支
3. `client/src/pages/RoomPage.tsx` — 添加 `chinese-chess` 路由
4. `client/src/pages/LobbyPage.tsx` — 创建房间配置 `rows: 10, cols: 9`
5. `server/src/websocket/Dispatcher.ts` — AI 走棋调度（返回 `{ from, to }`）

## 棋盘 UI 设计

- Canvas 绘制，仿木纹底色渐变
- 棋子用圆形 + 文字（红/黑配色，楷体）
- 棋盘线、楚河汉界、九宫格斜线
- 选中高亮（虚线圆）
- 上一步走法轨迹（原位白色虚线圆 + 新位绿色四角标记）
- 落子音效、悬停预览
- 将军音效、将死音效
- 将军文字提示

## AI 设计

- Minimax 搜索 + Alpha-Beta 剪枝
- 搜索深度: 1-4 层（对应四个难度）
- 评估函数: 棋子价值 + 位置价值表 + 将军威胁
- 棋子价值: 車=900, 砲=450, 馬=400, 仕=200, 相=200, 兵=100, 帅=10000
- 位置价值表: 每种棋子各一套 10×9 评估表

### AI 难度与人类水平对照

| 难度 | 搜索深度 | 大约 Elo 分数 | 人类水平 |
|------|---------|--------------|---------|
| 简单 | 1 层 | 800-1000 | 入门级 |
| 普通 | 2 层 | 1200-1400 | 俱乐部棋手 |
| 中等 | 3 层 | 1500-1700 | 中级棋手 |
| 困难 | 4 层 | 1800-2000 | 强俱乐部/候补大师 |

## 棋谱保存

- **先手**: 红方
- **存储位置**: `game_records` 表的 `pgn` 字段
- **棋谱格式**: PGN（中国象棋记谱法）
- **格式说明**: 使用中文棋子名 + 数字坐标，红方用中文数字（一-九），黑方用阿拉伯数字
- **示例**: `1. 炮二平五 马8进7 2. 馬二进三 车9平8`
