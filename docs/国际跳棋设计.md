# 国际跳棋实现设计

## 概述

按照围棋/五子棋/中国象棋/国际象棋的架构模式，在 `games/draughts/` 下实现完整的国际跳棋游戏，包括规则引擎、AI、棋盘组件和房间页面。

## 棋盘与规则

- **棋盘**: 10列 × 10行（row 0-9, col 0-9），只使用深色格子（row+col 为奇数）
- **白方先行**
- **棋子**: 白方（⚪兵/♛王）/ 黑方（⚫兵/♚王）

### 棋子走法规则

| 棋子 | 走法 |
|------|------|
| 兵 (Man) | 斜向前走一格（白方向上，黑方向下） |
| 王 (King) | 斜向前/后走一格（可后退） |

### 特殊规则

- **吃子**: 斜向跳过对方棋子落到空位，必须吃子（强制吃子）
- **多重吃子**: 吃子后如果还能继续吃，必须连续跳完
- **升变**: 兵到达对方底线（白方 row 0，黑方 row 9）升变为王
- **胜负**: 对方无子可走或被吃完则输

## 集成点

1. `shared/src/types/game.ts` — `Draughts = 'draughts'`
2. `server/src/websocket/Dispatcher.ts` — `getEngine()` 添加国际跳棋分支
3. `client/src/pages/RoomPage.tsx` — 添加 `draughts` 路由
4. `client/src/pages/LobbyPage.tsx` — 创建房间配置 `rows: 10, cols: 10`
5. `server/src/websocket/Dispatcher.ts` — AI 走棋调度（返回 `{ from, to }`）

## 棋盘 UI 设计

- Canvas 绘制，10×10 交替深浅色格（浅色 #f0d9b5，深色 #b58863）
- 只在深色格子上放置棋子
- 棋子用圆形渐变（白棋 #ffffff→#cccccc，黑棋 #555555→#1a1a1a）
- 王的标记（皇冠符号 ♛）
- 选中高亮、合法走法提示点
- 上一步走法轨迹高亮
- 落子音效
- 白方视角在下方，黑方视角自动翻转

## AI 设计

- Minimax 搜索 + Alpha-Beta 剪枝
- 搜索深度: 2-5 层（对应四个难度，去掉 depth 1 随机走棋）
- 评估函数: 棋子价值 + 位置价值表
- 棋子价值: 王=500, 兵=100
- 位置价值表: 10×10 评估表，中心控制加分

### AI 难度与人类水平对照

| 难度 | 搜索深度 | 人类水平 |
|------|---------|---------|
| 简单 | 2 层 | 入门级 |
| 普通 | 3 层 | 业余初级 |
| 中等 | 4 层 | 业余中级 |
| 困难 | 5 层 | 业余高级 |

## 棋谱保存

- **先手**: 白方
- **存储位置**: `game_records` 表的 `pdn` 字段
- **棋谱格式**: PDN（Portable Draughts Notation）
- **格式说明**: 使用深色格子编号 1-50，走棋用 `-`，吃子用 `x`
- **示例**: `1. 32-28 19-23 2. 28x19 14x23`
